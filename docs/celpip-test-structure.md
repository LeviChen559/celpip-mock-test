> Note: The Listening and/or Reading Tests may contain unscored content used for test development. This can be found anywhere within the Listening and/or Reading Tests and will have the same format as the scored content. You will not be able to tell scored content from unscored content, so apply your best effort to the entire test. Time spent on unscored content is included in the total test time and does not take time away from scored tasks.
